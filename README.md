# Gym Workout Tracker
A Python desktop application built with Tkinter for gym members to create personalized workout programs, track exercises, and save workout data.

## Features
- User profile creation (name, age, height, weight, fitness goals)
- Exercise logging with sets, reps, and weight tracking
- Workout file generation and management
- BMI calculation with unit conversion (lbs â†” kg)
- Workout analysis (max weight used, best set ratios)
- Interactive GUI with multiple windows

## Technologies Used
- Language: Python 3.x
- GUI Framework:Tkinter
- Concepts: Object-Oriented Programming, File Handling, Event-Driven Programming
